# 🎓 Student Academic Performance Prediction System

A Machine Learning-based web application that predicts whether a university student is likely to **Pass** or become **At-Risk** based on academic, behavioral, and learning-related factors.

The system was developed using **Python**, **Scikit-learn**, and **Streamlit**, with a Decision Tree Classifier trained on questionnaire data collected from undergraduate students.

---

## 📌 Project Objectives

The objectives of this project are to:

- Predict students' academic performance using machine learning.
- Identify students who may be academically at risk.
- Support educators and institutions in making early intervention decisions.
- Provide an easy-to-use web application for academic performance prediction.

## 📊 Dataset

The dataset consists of questionnaire responses collected from undergraduate students and includes features such as:

- Gender
- Age
- Level of Study
- Faculty
- Department
- Current CGPA
- Continuous Assessment (CA) Score
- Class Attendance
- Study Hours
- Social Media Usage
- Academic Internet Research Hours
- Learning Device
- Online Educational Resource Usage
- Time Management
- Assignment Completion
- Online Discussion Participation

Target Variable:

- Prediction Class (Pass / At-Risk)



Install dependencies:

```bash

```
